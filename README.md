# 🎨 Derpixon Fan Site

> *"Unleashing Art Without Boundaries"*

A fan-made tribute site dedicated to **Derpixon** — one of the most imaginative and distinctive animation creators on the internet. This project is a love letter to their work, built from scratch as a personal project by a genuine fan.

---

## 💡 About the Project

This is a single-page fan site built in pure HTML and CSS. No frameworks, no build tools — just hand-crafted markup and styling that channels the bold, striking visual identity of Derpixon's art style. The whole thing runs in a compact 500px canvas that scales responsively to any screen size using a simple zoom trick.

The site was made because Derpixon's work genuinely deserves more spotlight. If you've ever seen their animations, you'll know exactly why.

---

## ✨ Features

- **Custom header** with a signature demon silhouette motif and Derpixon's iconic red-and-black colour palette
- **Links to all official channels** so fans can easily find and support Derpixon in one place
- **Fan art credits section** at the bottom of the page, acknowledging the community that rallies around the creator
- **Fully responsive scaling** — looks clean on any screen width without breaking layout

## 🛠️ Built With

- Plain **HTML5**

No dependencies. No npm install. Just open the file and it works.

## 🚀 Getting Started

1. Clone or download the repo
2. Open `derpixon-site.html` in your browser
3. That's it — you're done

```bash
git clone https://github.com/your-username/derpixon-fan-site.git
cd derpixon-fan-site
open derpixon-site.html
```
